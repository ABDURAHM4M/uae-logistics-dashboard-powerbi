# UAE Logistics Operations Dashboard (Power BI)

![Dashboard Preview](ABBB%20report-template-with-data/dashboard%20Image%20.jpeg)

## Project Overview

This project analyzes logistics shipment performance across UAE transport routes using Power BI.

The dashboard monitors operational KPIs including shipment volume, delivery delays, logistics costs, and route efficiency.

## Key Metrics

- Total Shipments
- Delay Rate %
- On-Time Delivery Rate
- Total Revenue (AED)
- Average Cost per KM
- Average Delivery Duration

## Insights

The dashboard enables logistics analysts to:

- Track shipment performance trends
- Monitor delivery delays across transport modes
- Evaluate revenue trends and route performance
- Identify operational bottlenecks in logistics operations

## Tools Used

- Power BI
- DAX
- Data Modeling
- Logistics KPI Analysis

## Dataset

The dataset contains shipment logistics data including:

- transport type
- origin and destination routes
- shipment cost
- delivery duration
- weather conditions
- delay indicators

## Business Value

This dashboard helps logistics teams monitor operational performance, identify delays, evaluate route profitability, and improve supply chain decision making.
